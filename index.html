embedded_url

Unobtrusive embedded HTML elements for Ruby

When provided with an address, Embedded URL will attempt to generate markup for an embedded representation of the URL. This markup may be inserted directly into your webpage without the hassle of dealing with ugly <embed> and <object> tags.

Dependencies

  • httparty (for SlideShare support)

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/dismaldenizen/embedded_url

get the source code on GitHub : dismaldenizen/embedded_url
